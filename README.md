## üöÄ Quick Start

### Prerequisites
- A web browser (Chrome, Firefox, Safari, Edge)
- Python 3.x (for local server) or any static file server

### Local Development

1. **Navigate to the project directory:**
   ```bash
   cd 'C:\Users\ABEY\Downloads\web\me'
   ```

2. **Create assets folder (if needed):**
   ```bash
   mkdir assets
   # Place your logo.png and other images in the assets folder
   ```

3. **Serve the files locally:**
   ```bash
   python -m http.server 8000
   ```

4. **Open in browser:**
   Navigate to `http://localhost:8000/abi.html`

### Alternative: Direct File Opening
- Double-click `abi.html` in your file explorer
- Or open via `file://` protocol in your browser
- *Note: Serving over HTTP is recommended to avoid CORS issues with CDN assets*

## üîß Customization

### Changing the Logo
1. Place your logo image in the `assets/` folder as `logo.png`
2. The HTML automatically references `assets/logo.png`

### Updating Content
- **Personal Information**: Edit the About section in `abi.html`
- **Portfolio Items**: Modify the portfolio grid in the Portfolio section
- **Services**: Update service descriptions and features
- **Contact Details**: Change email, phone, and location in the Contact section

### Styling Modifications
- Colors are defined in the Tailwind config within the HTML head
- Custom CSS is included for animations and specific components
- Modify the `style` tag in the HTML for custom styling

### Adding New Sections
- Follow the existing section structure
- Use Tailwind classes for consistent styling
- Update navigation links accordingly
